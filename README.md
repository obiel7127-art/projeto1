O projeto tem um index.html que mostra uma imagem de lâmpada e dois botões: Ligar e Desligar.

O arquivo lampada.js troca a imagem da lâmpada para acesa ou apagada quando os botões são clicados.

O style.css pode ser usado para ajustar o visual (centralizar, tamanho da imagem etc.).

Para usar: coloque todos os arquivos na mesma pasta e abra o index.html no navegador.
